Select an audio file to continue. Remember to add some silence to the start of your file so the chart doesn't begin right away.

After selecting a file you can either upload a chart to edit, or use the tap input to begin creating a new chart

Planned Features:

- Load a *.chart file into the app

- Change song name/artist/charter in the app

- Sustain notes

- BPM changes mid song

- Time signatures other than 4/4

- Tap notes

- Star power notes

- Solos

- Sections

- View / Edit hammer ons and pull offs

- Adjust chart preview speed

- Fast forward and rewind in chart preview

- Multiple audio files
